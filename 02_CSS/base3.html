Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагаеся слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1
Размер блока:
Занимает доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ от соседней равным 0.2 em, при помощи
свойства margin-bottom

Статья 2 Размер блока:
Занимает доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ от соседней равным 0.2 em, при помощи
свойства margin-bottom. Каринка вставлена при помощи тега, плавание
организовано по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал.Занимает всю доступную ширину.Высота определяется помещенным внутри контентом.
